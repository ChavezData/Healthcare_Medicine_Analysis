📊 Medicine Review Analysis

A data analysis project to explore and visualize user feedback on medicines using Python and Jupyter Notebook.

This repository contains a Jupyter Notebook (Healthcare_Medicine.ipynb) that performs exploratory data analysis (EDA) on a dataset of medicine reviews to understand patterns in user feedback and identify medicines with the highest and lowest satisfaction levels.  ￼

⸻

🧠 Project Overview

The goal of this project is to:
	•	📈 Analyze user reviews of medicines from a structured dataset.
	•	🔍 Quantify perceptions using percentages of Excellent, Average, and Poor reviews.
	•	🧹 Clean and preprocess the data for meaningful analysis.
	•	📊 Visualize trends and highlight insights into medicine reviews.

⸻

📋 What’s Included

File	Description
Healthcare_Medicine.ipynb	Main analysis notebook performing EDA, cleaning, visualization, and insights.
Medicine_Details.csv (expected)	Dataset containing medicine information and review statistics.


⸻

🚀 Features

✔ Load and inspect raw medicine review data
✔ Data cleaning and type conversion of key numeric columns
✔ Top-reviewed vs. poorly reviewed medicine identification
✔ Visual distribution of review categories
✔ Insights into review patterns for manufacturers and medicine types

📦 Installation

To run this project locally:
	1.	Clone the repository

git clone https://github.com/ChavezData/Healthcare_Medicine_Analysis.git
cd Healthcare_Medicine_Analysis

	2.	Install dependencies

Make sure you have Python 3 installed.

pip install pandas numpy matplotlib seaborn

	3.	Launch the Notebook

jupyter notebook Healthcare_Medicine.ipynb


⸻

🧪 Usage

Inside the notebook you will find steps to:
	1.	Load the CSV dataset into a DataFrame.  ￼
	2.	Inspect the first few records to understand structure and content.  ￼
	3.	Convert review percent columns to numeric types for analysis.  ￼
	4.	Create visualizations (e.g., histograms, bar charts) to explore distributions.
	5.	Derive insights on medicines that receive consistently high or low user reviews.
